Enable New Product Icons per Organization
-----------------------------------------

You can mark certain selected products to be displayed as new ones in the front store.

.. begin

To enable New Product icons per organization:

1. Navigate to **System > User Management > Organizations** in the main menu.
2. For the necessary organization, hover over the |IcMore| **More Options** menu to the right of the necessary organization and click the |IcConfig| **Configuration** icon to start editing the configuration.
3. Select **Commerce > Product > Promotions** in the menu to the left.

.. note::
   For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

The following page opens.

.. image:: /configuration_guide/img/configuration/product/new_arrivals/NewArrivalsBlockOrg.png

4. In the **New Product Icons**, clear the **Use Default** check box and select *Yes* in the **Show on Product View** list.

4. Click **Save**.

.. note:: To disable the 'New Product' icons, select *No* in the **Show on Product View** list, and click **Save**.

.. finish
