Enable Mass Product Actions per Website
---------------------------------------

.. begin_body

To enable mass product actions per website:

1. Navigate to **System > Websites** in the main menu.
2. For the necessary website, hover over the |IcMore| **More Options** menu to the right of the necessary website and click |IcConfig| to start editing the configuration.
3. Select **Commerce > Sales > Shopping List** in the menu on the left.

.. note::
   For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

The following page opens:

.. image:: /user_guide/img/system/configuration/sales/shopping_list/MassProductActionsSListWeb.png

4. In the **Shopping List Options** section, set whether customer users are allowed to mass select and add items to the shopping list.

   By default, mass product actions are enabled.

   To disable them, clear the *Use System* and the *Enable Mass Adding on Product Listing* check boxes.

5. Click **Save Settings**.

.. finish_body
