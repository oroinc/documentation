Configure Order Creation per Organization
-------------------------------------------

.. begin_body

You can set the following order creation related options for the organization to override global configuration:

.. include:: /user_guide/system/commerce_configuration/orders/order_creation.rst
   :start-after: start_order_creation
   :end-before: stop_order_creation

To configure the order creation options per organization:

1. Navigate to **System > User Management > Organizations** in the main menu.
2. For the necessary organization, hover over the |IcMore| **More Options** menu at the end of the row and click |IcConfig| to start editing the configuration.

   .. note:: For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

3. In the **Order Creation** section:

   a) Clear the **Use System** check box if it is selected.
   b) From the **New Internal Order Status** list, select the required status.

#. Click **Save Settings**.
