Configure Order Automation Globally
-----------------------------------

.. begin_body

You can set the following order automation related options that apply globally by default:

.. include:: /user_guide/system/commerce_configuration/orders/order_automation.rst
   :start-after: start_order_automation
   :end-before: stop_order_automation


To configure the order automation options globally:

1. Navigate to **System > Configuration** in the main menu.
2. Select **Commerce > Orders > Order Automation** in the menu to the left.

.. note:: For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

3. In the **Order Automation** section:

   a) Clear the **Use System** check box.
   b) Select the **Enable Automatic Order Cancellation** check box.


#. Click **Save Settings**.
