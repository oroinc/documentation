Enable New Product Icons per Website
------------------------------------

You can mark certain selected products to be displayed as new ones in the front store.

.. begin

To enable New Product icons per website:

1. Navigate to **System > Website** in the main menu.
2. For the necessary website, hover over the |IcMore| more actions menu to the right of the necessary website and click |IcConfig| to start editing the configuration.
3. Select **Commerce > Product > Promotions** in the menu to the left.
4. In the **New Product Icons**, clear the **Use Default** box and select **Yes** in the **Show on Product View** field.

   The following page opens.

.. image:: /user_guide/img/system/configuration/product/new_arrivals/NewArrivalsPerWebsite.png
   :class: with-border

4. Click **Save**.

.. note:: To disable New Product icons, select **No** in the **Show on Product View** field, and click **Save**.

.. finish
