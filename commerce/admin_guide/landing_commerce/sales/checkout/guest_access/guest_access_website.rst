Enable Guest Website Access per Website
---------------------------------------

.. begin

To enable guest access per website:

1. Navigate to **System > Websites**.
2. For the necessary website, hover over the |IcMore| **More Options** menu to the right of the necessary website, and click |IcConfig| to start editing the configuration.
3. Select **Commerce > Guests > Website Access** in the menu on the left.

.. note::
   For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

4. Select the **Enable Guest Access** checkbox.
5. Click **Save Settings**.
   
.. note:: To disable guest access globally, clear the **Enable Guest Access** checkbox.

.. finish
