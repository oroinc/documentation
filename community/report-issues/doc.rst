Report a Documentation Issue
============================

To report an issue in the Oro documentation, file a suggestion for improvement, or share your feedback, you can:

* Send an email to `doc@oroinc.com <mailto:doc@oroinc.com>`_.
* Message us on |Twitter|.
* :ref:`Submit an issue <doc--community--issue-report>` to our GitHub repository.

You are also welcome to contribute to the Oro documentation. To learn how, please see the :ref:`Contribute to Documentation <documentation-standards>` topic.
