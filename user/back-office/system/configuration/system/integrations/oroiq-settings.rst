Configure Global OroIQ Settings
===============================

.. hint:: Please |contact our support team| to learn more about OroCommerce AI features, discuss how they can meet your business needs, and get started with implementation.

:ref:`OroIQ <concept-guide--oroiq>` is an AI-powered assistant designed to help users analyze data and perform back-office operations in the OroCommerce Enterprise application using natural language. Once OroIQ assistant is installed in the Oro application, it can be enabled globally and per :ref:`organization <organization-oroiq-settings>`.

.. image:: /user/img/concept-guides/ai/oroiq-icon.png

1. Navigate to **System > Configuration** in the main menu.
2. In the menu to the left, click **System Configuration > Integrations > OroIQ**.
3. In the **OroIQ** section, clear the **Use Default** checkbox next to *Enable OroIQ* to enable the assistant. To use OroIQ, you must first enable OAuth2 by configuring encryption keys. For the best experience, we also recommend enabling the following system option :ref:`Commerce > Search > Fuzzy Search > Enable Fuzzy Search <configuration--guide--commerce--configuration--fuzzy-search>`.
4. Agent API Key and Agent URL are provided by the Oro Team.
5. Click **Check Connection** to verify that the configuration is valid.
6. Click **Save Settings**.
