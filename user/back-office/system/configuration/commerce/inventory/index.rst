Inventory
=========

.. important:: This section is a part of the :ref:`Inventory and Warehouse Management <concept-guide--inventory>` topic that provides the general understanding of the inventory and warehouse concepts.

Discover how to configure your storefront and back-office inventory statuses, add warehouses, manage product inventory options, set inventory thresholds and quantity limitations. You can control the way inventory is managed in OroCommerce on several levels -- globally, :ref:`per organization <configuration--commerce--inventory-organization>`, :ref:`per website <configuration--commerce--inventory-website>`, :ref:`per product <create-simple-product-inventory>`, and :ref:`per master catalog category <master-catalog-inventory>`.
