Configure Global Order Automation Settings
==========================================

You can set all orders with selected internal statuses to be automatically cancelled past their "Do Not Ship Later Than" date.

.. hint:: You can enable/disable automatic order cancellation globally and :ref:`per organization <configuration--commerce--orders--order-automation--organization>`.

To configure automatic order cancellation globally:

1. Navigate to **System > Configuration** in the main menu.
2. Select **Commerce > Orders > Order Automation** in the menu to the left.

.. note:: For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

3. In the **Order Automation** section:

   a) Clear the **Use Default** checkbox.
   b) Select the **Enable Automatic Order Cancellation** checkbox.

#. Click **Save Settings**.
